How It Works
Eleven pipelines. Zero babysitting.
From CRM sync to booked meeting — the agent runs every step. Your sales team wakes up to warm conversations, not empty inboxes and manual tasks.
01

Syncs Your CRM

Pulls contacts, companies, and full engagement history from HubSpot or CSV every hour — emails, calls, meetings, notes, tasks, deals. Always fresh, zero manual entry.

02

Scores Every Account

AI scores each company 0–100 against your exact ICP. Funding round just announced? Hiring surge? New product launch? Those accounts jump to the top of the queue.

03

Researches Hot Accounts

Searches the web for real angles — competitive intel, leadership changes, press mentions. Turns generic outreach into emails prospects actually respond to.

04

Discovers the Right Contacts

Finds decision-makers at high-scoring accounts via Apollo. Enriches with phone, LinkedIn, title, and seniority — no manual prospecting spreadsheets.

05

Coordinates Multi-Contact Strategy

Multiple contacts at the same company? The agent prevents carpet bombing, adjusts tone for engaged accounts, and blocks outreach during bad timing — all automatically.

06

Writes Emails That Convert

Generates 3-email sequences referencing specific facts about each prospect — not "Hi {first_name}." Deep personalization, enforced brand voice, AI self-evaluated before send.

07

Manages Sequences Automatically

Sends with smart timing, round-robin sender rotation, and daily limits. Named cadences (aggressive, standard, enterprise) auto-selected by ICP score. Stops the moment they reply.

08

Handles Every Reply

Classifies intent in seconds — interested, question, not interested, referral — and acts. Urgent Slack alert for hot leads. Opt-outs honored instantly. Nothing falls through the cracks.

09

Executes Human + AI Tasks

Your reps create tasks in plain English. The AI reads full context, decides what to do, and acts — or escalates with a clear reason when it should.

10

Monitors Its Own Health

Automated health checks every 15 minutes. Daily Slack digest with pipeline stats, reply rates, and anything needing attention — before you think to ask.

11

Reports Every Week

Performance summaries post to Slack every Friday. Meetings booked, reply rates, pipeline health — your week in numbers, no manual reporting.

Task Executor
Tell it what to do. In plain English.
Create a task for any lead. The agent reads everything it knows — past emails, signals, purchases, history — and decides what to do. No templates. No rigid workflows. Just context and instructions.

Execute

Has context, task is actionable. Sends email, adds note, or pings Slack.

Decline

Blocker exists or needs human judgment. Creates escalated task with reason.

Reschedule

Right task, wrong timing. Re-creates the task with a new due date.

Skip

Already done, duplicate, or irrelevant. Marks complete with reason.

Modular Pipeline Architecture

Every pipeline is a standalone, single-responsibility TypeScript file. sync-hubspot.ts does CRM sync. generate-outreach.ts writes emails. No pipeline knows about the others. Swap one, the rest keep running.

Fully Typed End to End

Strict TypeScript across 30+ source files. Every data model is defined with explicit interfaces. No any types. No runtime surprises.

One Config File

All technical settings live in prospecting.config.ts — target titles, sequence timing, enrichment limits, Gmail senders, rate limits, budget caps.

Self-Documenting for AI

Structured so AI tools (Cursor, Claude Code, Copilot, Windsurf) can reason about it effectively. Clear file names, types describe data shapes, explicit boundaries.

Getting Started
Clone to first AI email in 30 minutes.
# 1. Clone the repo
git clone https://github.com/personizeai/ai-prospecting-agent.git
cd ai-prospecting-agent

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env
# Fill in your API keys

# 4. Authorize Gmail (opens browser for OAuth2 consent)
npm run gmail:auth

# 5. Create Personize schemas and governance rules
npm run setup

# 6. Test locally (dry-run mode — no emails sent)
npx trigger.dev@latest dev

# 7. Deploy to production
npx trigger.dev@latest deploy

Going Live

1Review 20+ dry-run emails in the Trigger.dev dashboard
2Adjust governance rules in Personize if tone needs tuning
3Set DRY_RUN=false in your environment
4Redeploy and monitor Slack for the first few days
